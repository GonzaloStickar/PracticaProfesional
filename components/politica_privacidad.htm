Política de Privacidad y Seguridad de Datos

Introducción

Trussi Videos, valoramos y respetamos tu privacidad. Esta política de privacidad explica cómo recopilamos, utilizamos, y protegemos tus datos personales en cumplimiento con las leyes de protección de datos aplicables.

Datos Recopilados

Recopilamos datos personales, específicamente nombre y/o apellido, cuando los clientes utilizan nuestro servicio para consultar el estado de sus reparaciones. Estos datos son necesarios para identificar y proporcionar la información correcta sobre el estado de la reparación.

Uso de los Datos

Los datos personales recopilados se utilizan exclusivamente para:

  • Proporcionar información precisa sobre el estado de las reparaciones.

No utilizamos, compartimos, ni vendemos tus datos personales para otros fines.

Seguridad de los Datos

Implementamos medidas de seguridad robustas para proteger tus datos personales. Utilizamos métodos de hasheo y un certificado SSL proporcionado por Vercel para asegurar que los datos transmitidos a través de nuestro sitio web están protegidos contra accesos no autorizados.

Ley 25.326

Esta política se rige bajo la Ley 25.326 sobre Protección de los Datos Personales en Argentina. Nos comprometemos a cumplir con esta ley y a tratar tus datos personales con el máximo respeto y confidencialidad.

Modificaciones a esta Política
